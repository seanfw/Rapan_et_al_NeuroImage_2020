This repo contains code for analysis and visualisation of data from
Rapan, Lucija, Sean Froudist-Walsh, Meiqi Niu, Ting Xu, Thomas Funck, Karl Zilles, and Nicola Palomero-Gallagher.
"Multimodal 3D atlas of the macaque monkey motor and premotor cortex." NeuroImage (2020): 117574.

Code by Sean Froudist-Walsh (NYU)
Receptor data provided by Rapan, Niu, Zilles and Palomero-Gallagher.
Functional MRI data from PRIME-DE, preprocessed and provided by Ting Xu.
